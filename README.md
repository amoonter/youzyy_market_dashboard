# youzyy_market_dashboard

这是一个面向 A 股交易复盘与盘中决策的市场驾驶舱项目，核心重点放在两部分：`行情分析` 和 `选股系统`。

## 行情分析

行情分析模块不是只看指数涨跌，而是把市场放在一个更完整的交易框架里观察，重点包括：

- 大盘位置、支撑位、压力位
- 市场环境强弱与风险偏好
- 活跃市值变化，用来判断资金风格是否偏进攻
- 主线强度和板块轮动，识别当前资金最认可的方向
- 新闻消息、热榜、集合竞价、涨停梯队、板块情绪等多维信息汇总

这部分的目标，不是给出一句简单的看多或看空，而是回答几个更关键的问题：

- 当前市场适合进攻还是防守
- 资金主要集中在哪些方向
- 哪些热点只是脉冲，哪些方向可能演化成持续主线
- 当下更适合低吸、突破，还是先观察等待确认

## 选股系统

选股系统从沪深全市场中筛选交易机会，不依赖单一形态，而是把多种因子放在一起综合判断。

目前选股会综合考虑：

- 形态结构，如趋势回踩、突破确认、关键 K 线、平台/箱体结构、老鸭头、杯柄等
- 技术指标，如 EMA、ADX、MACD、KDJ、BOLL、RSI、OBV、量能与动量
- 主线属性，判断个股是否属于当下资金认可的核心方向
- 热榜与人气，观察市场关注度和资金参与广度
- 六维分析，包括趋势动能、资金成分和质量、估值安全、基本面、周期位置、事件风险
- 机构与研究信息，用来增强对股票质量和一致预期的判断

系统最终会输出更接近交易执行的信息，而不只是“推荐股票名单”，包括：

- 入选原因
- 触发条件
- 失效条件
- 支撑位与压力位
- 仓位建议

这样做的目的，是把“为什么选中”和“什么时候可以做、什么时候不能做”讲清楚，方便盘中执行，而不是只给静态结果。

## 投研分析

投研部分会围绕主线板块继续拆到细分产业链，按上游、中游、下游展示轮动结构，并结合概念板块、细分环节、成交量、轮动状态和代表个股，辅助判断主线内部的扩散与接力关系。

## 自动更新与发布

项目支持按多个时间点自动刷新页面，计划内更新会自动同步到 GitHub Pages，因此本地页面和线上页面可以持续保持更新。
